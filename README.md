# 郑州信息科技职业学院 · 专业推荐系统

> 输入高考分数，智能匹配推荐专业，查看学习内容、就业率与就业方向。

## 功能

- **分数输入**：输入高考总分（0-750），选择科类（理科/文科）
- **兴趣选择**：从 5 个大类（信息技术、电子机械、财经管理、建筑工程、人文教育）中选择兴趣方向
- **录取概率**：基于近 3 年录取数据估算学校和专业的录取概率
- **专业推荐**：展示 20+ 个专业的详细信息，包括主干课程、就业率、就业方向、薪资参考
- **排序筛选**：可按录取概率、就业率排序，按兴趣方向筛选

## 数据说明

本系统数据基于郑州信息科技职业学院历年公开的招录信息编译。录取概率采用简化模型估算，**仅供志愿填报参考**。实际录取结果受当年报考人数、试题难度、招生计划变动等多因素影响，请结合河南省教育考试院官方信息综合判断。

## 技术栈

- 纯前端：原生 HTML + CSS + JavaScript
- 零依赖、无构建步骤
- 响应式设计，适配手机、平板、桌面
- 可部署在 GitHub Pages / Netlify / Vercel 等静态托管平台

## 部署方法

### GitHub Pages

1. 在 GitHub 新建仓库（如 `zzxvit-major-recommend`）
2. 将本目录所有文件推送到仓库
3. 进入仓库 Settings → Pages → 选择 main 分支
4. 等待部署完成，访问 `https://<你的用户名>.github.io/<仓库名>`

### 本地预览

```bash
cd major-recommend
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```
